# LabMidnight

> 加班累了是吗 —— 一款基于 PyGame 的第一人称伪 3D 恐怖解谜 RPG

## 简介

凌晨两点，你在实验楼中醒来，准备离开返回寝室，却发现实验楼突然停电，走廊陷入黑暗，手机信号异常。一间本应空无一人的教室中传出老师讲课的声音，某个角落传来奇怪的笑声……你需要在黑暗中探索实验楼，收集关键物品，破解谜题，最终找到出口并逃离。

## 游戏特色

- **伪 3D 渲染**：基于 Raycasting 射线投射算法，将二维地图渲染为第一人称视角
- **恐怖氛围**：暗色调画面、手电筒照明、音效事件层层递进
- **轻量解谜**：实验室脱出、异常教室线索、配电室恢复供电等谜题
- **资源管理**：手电电量限制视野，理智值影响画面与音效
- **多结局设计**：成功逃离 or 被困循环，取决于玩家的选择与表现

## 操作说明

| 按键 | 功能 |
|------|------|
| W / S | 前进 / 后退 |
| A / D | 左转 / 右转 |
| F | 交互（开门、拾取道具、查看线索） |
| 空格 | 跳跃 |
| 鼠标左键 | 使用当前道具 / 确认操作 |
| 鼠标右键 | 使用快捷工具 |

## 玩法流程

1. **实验室脱出** — 醒来后在实验室内寻找钥匙或线索离开
2. **走廊探索** — 循着讲课声和线索在黑暗中前进
3. **异常教室** — 发现无人教室中的黑板和纸条隐藏配电室密码
4. **恢复供电** — 收集保险丝，前往配电室输入密码恢复电力
5. **逃离出口** — 取得门禁卡，打开出口门禁，逃离实验楼

## 状态系统

- **生命值**：受到异常事件或陷阱影响后的生存状态
- **理智值**：长时间处于黑暗或遭遇异常音效时下降，影响画面与音效表现
- **手电电量**：随使用时间消耗，电量不足时可见距离缩短，需收集电池补充

## 技术栈

- **语言**：Python
- **游戏引擎**：Pygame
- **核心算法**：Raycasting 射线投射伪 3D 渲染
